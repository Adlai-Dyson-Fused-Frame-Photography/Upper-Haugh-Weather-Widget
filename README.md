Upper Haugh Weather Widget
-----

A widget, perfect for displying on a dashboard, slideshow, or kiosk, for Upper Haugh, Rawmarsh, Rotherham

----------

*   **Shows Location** : Upper Haugh

*   **Shows temperature in Celsius**
